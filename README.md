## Раскраска графа </br>
Проект содержит реализацию трёх алгоритмов раскраски графа: </br>
- Жадный алгоритм </br>
- Жадный оптимизированный </br>
- Полного перебора (для каждой перестановки вершин выполняется жадный алгоритм)</br>
